This program is a multithreaded implementation of calculating Pi 
using a Monte Carlo simulation.  To compile, run the following command
within in folder:

javac MonteCarlo.java

The program is run by using the command:

java MonteCarlo X

Where X is an OPTIONAL integer specifying the number of points you'd like the simulation
to generate. If no argument is provided, then the default value of 10,000 is used.
